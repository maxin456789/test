Скрипт записаная книжка
Для запуска необходимо создать файл phonebook.csv в корневой папке
Далее запустить main.py

Возможности программы:
0 - сбросить изменения
1 - показать все контакты
2 - создать контакт
3 - найти контакт
4 - изменить контакт
5 - удалить контакт
6 - сохранить изменения
7 - выход
